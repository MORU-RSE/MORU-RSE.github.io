## Overview

EpiModel is an epidemiological simulation platform built to support infectious disease research across multiple institutions. The platform provides an open-access environment for sharing de-identified case data, running models, and publishing reproducible results.

## Technical Stack

- **Backend**: Python / FastAPI
- **Frontend**: React + D3.js for interactive visualisation
- **Data**: PostgreSQL with PostGIS for geospatial queries
- **Infrastructure**: Docker / Kubernetes on cloud

## Impact

The platform has supported over 30 peer-reviewed publications and is used by research groups across 12 countries.

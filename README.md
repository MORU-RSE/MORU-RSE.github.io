# MORU RSE Webpage

This repository contains the source code for the MORU Research Software Engineering (RSE) webpage.

Website: https://moru-rse.github.io/

The MORU RSE webpage is designed to make research software engineering activities more visible, organized, and accessible.

It helps visitors understand what the RSE team does, what projects are being developed, and how research software contributes to scientific work. It also acts as a reference point for team members, collaborators, and researchers who want to learn more about available tools, ongoing work, and possible areas of collaboration.

---

## Purpose

The purpose of this website is to provide a clear and accessible platform for sharing information about the MORU RSE unit.

The website aims to:

- Introduce the RSE team and its role within MORU
- Showcase research software projects and outputs
- Share software tools, resources, and documentation
- Publish news, updates, and announcements
- Present publications and related research outputs
- Provide information about opportunities and collaborations
- Support communication between the RSE team, researchers, and external partners
